1. 문자열 -> string -> text
2. 문장 -> text -> text
3. 정수 -> integer-> integer
4. 실수(소숫점포함) -> real-> real
5. 참/거짓 -> boolean -> ?
6. 날짜 -> ymd -> date
7. 날짜시간->datetime-> datetime ?
