# 설계

## step1에서의 data_type

- step1은 `create table` sql 문장을 수행할 수 있게 json을 만든다.

## data_type 설계

UI -> json -> sqlite column type

1. 문자열 -> string -> text
2. 문장 -> text -> text
3. 정수 -> integer-> integer
4. 실수(소숫점포함) -> real-> real
5. 참/거짓 -> boolean -> ?
6. 날짜 -> ymd -> date
7. 날짜시간->datetime-> datetime ?

- constants folder 의 data_type.py에 class로 정의하여 사용함

## element_type

- 정해진 element_type으로 각 기록을 create, edit 할 때  적절한 html을 만들 수 있다.

UI -> json -> html

1. 문자열 -> input-text -> `input type=text`
2. 문장 -> input-html -> `textarea`
3. 문장(포맷)->html-editor-> `quill등적용`
4. 날짜 -> input-date -> `input type=date`
5. 숫자(정수) -> input-integer -> `input type=number`
6. 숫자(소숫점 포함) -> input-real-> `input type=number`
7. 이메일 -> input-email -> `input type=email`
8. 1개선택(radio) -> radio-> `input type=radio`
9. 여러개 선택 -> check-box-multi -> `checkbox`
10. 체크박스-> check -> `checkbox`
11. 이미지선택-> image-upload -> `file image`
12. 파일 선택-> file-upload-> `file`
13. 드롭박스-> dropdown -> `select`

- constants 폴더의 element_type.py에 정의를 함

## display_type

- list 나 상세보기에서 각 필드를 어떻게 보여줄 지 정한다.
- 부가적으로 format이 있을 수 있다.
  
UI -> json -> html

6. 파일링크 -> file -> <a href="..." download>📎 파일명.pdf</a>

## 기본 표시

1. 텍스트 -> text -> <span>일반 텍스트</span>
2. HTML -> html -> <div class="html-content">렌더링된 HTML</div>
3. 여러줄 텍스트 -> multiline -> <pre class="whitespace-pre-wrap">...</pre>

## 숫자 표시

1. 숫자(콤마) -> number_comma -> <span>1,234,567</span>
2. 통화 -> currency -> <span>₩1,234,567</span>
3. 백분율 -> percentage -> <span>85.5%</span>

## 날짜/시간 표시

1. 날짜 -> date -> <span>2024년 1월 1일</span>
2. 시간 -> time -> <span>14:30</span>
3. 날짜+시간 -> datetime -> <span>2024-01-01 14:30</span>
4. 상대시간 -> relative_time -> <span>3일 전</span>

## 불린/상태

1. 불린(O/X) -> boolean -> <span>✓</span> 또는 <span>✗</span>
2. 불린(뱃지) -> badge -> <span class="badge-success">활성</span>
3. 상태 -> status -> <span class="status-dot-green">진행중</span>

## 특수 표시

1. 별점 -> stars -> <span>★★★★☆</span>
2. 프로그레스바 -> progress -> <div class="progress-bar" style="width:75%">
3. 컬러칩 -> color -> <span class="color-chip" style="background:#FF5733">
4. 이미지 -> image -> <img src="..." class="thumbnail">
5. 링크 -> link -> <a href="...">클릭</a>
6. 이메일 -> email -> <a href="mailto:...">email@example.com</a>

## 목록

1. 태그목록 -> tags -> <span class="tag">태그1</span> <span class="tag">태그2</span>
2. 쉼표구분 -> comma_list -> <span>항목1, 항목2, 항목3</span>
3. 불릿목록 -> bullet_list -> <ul><li>항목1</li><li>항목2</li></ul>

## 파일

1. 파일링크 -> file_link -> <a href="..." download>파일명.pdf</a>
2. 이미지갤러리 -> image_gallery -> <div class="gallery">...</div>
3. 파일목록 -> file_list -> <ul class="file-list">...</ul>

## 코드/데이터

1. 코드블록 -> code -> <pre><code>...</code></pre>
2. JSON -> json -> <pre class="json-viewer">...</pre>
